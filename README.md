#### 1.html5-qrcode
**功能：**
**1.1** h5页面在微博客户端中呼起摄像头扫描二维码并且解析。
**1.2** h5页面在非微博客户端中（原生浏览器或者微信客户端）呼起系统拍照或者上传图片按钮，拍照二维码或者上传二维码并且解析
具体使用方法，参看**[说明文档](https://github.com/zhiqiang21/WebComponent/tree/master/html5-Qrcode/README.md)**

**预览：**
